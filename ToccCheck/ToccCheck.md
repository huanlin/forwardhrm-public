# 安裝

下載之後，直接開啟並執行安裝程式，一直點「下一步」應該就能完成安裝。
安裝完成後，桌面上會多一個捷徑：

![image](https://user-images.githubusercontent.com/605065/129508052-49cf6069-a4e7-4e1b-9c56-43819a6ab238.png)

從 Windows 的程式集選單裡面也找得到一個名為「TOCC 檢核程式」的捷徑。
